## 二、哨兵(兔子型 A,教學基準)

**一句話定位**:輕甲低需求偵察兵,教學基準,定義「弱點對應 vs 不對應」的次數差距感

**敵人類別**:**兔子(A)** — 無威脅普通,失敗時逃走,不會精英化回歸

### 層 1 — 模板永久顯性

- 大類識別:輕甲偵察
- 敵人類別:**兔子**
- 弱點範圍:衝擊 / 穿刺(燃燒不在範圍——林地偵察兵不能用火惹眼;只能透過混合任意路徑計入)
- 弱點需求層級範圍:1(低)/ 2(中低)
- 戰鬥狀態傾向:預設標準戰

### 層 2 — 次級視覺線索

> 對應 [敵人設計原則.md 第四章](敵人設計原則.md) 視覺線索詞彙表。

- 持弓/弩 → 弱點衝擊(對應 A.1 — 遠距武器使用者重心輕盈,慣於保持距離輸出,近身衝擊難以即時穩定重心)
- 持短劍 + 輕盾 → 弱點穿刺(對應 A.2 — 輕盾保護面積有限,細尖物可繞盾邊或從盾下進入)
- 殘破皮甲 / 瘦小體型 → 弱點需求 1(低,易破)
- 完整皮甲 / 一般體型 → 弱點需求 2(中低)
- **兔子型視覺**:小體型、無威脅姿態、武器簡陋(對應 D.1)

**覆蓋承諾**:每個實例必持「弓/弩」或「短劍 + 盾」其中之一,絕不出現無弱點視覺暗示的實例。

### 層 3 — 即時狀態

- 戰鬥狀態標籤
- 上一場用情報牌揭露的相關資訊

### 層 4 — 隱藏狀態(僅情報牌可揭)

完整需求表(從以下 4 個 instance 之一):
- **A 弱衝擊低**:衝擊 1 / 穿刺 4 / 混合任意 4
- **B 弱衝擊中低**:衝擊 2 / 穿刺 5 / 混合任意 8
- **C 弱穿刺低**:衝擊 4 / 穿刺 1 / 混合任意 4
- **D 弱穿刺中低**:衝擊 5 / 穿刺 2 / 混合任意 8

### 實例變異規則

此模板刻意降低變異複雜度,作為教學基準:
- 100% — 主軸:需求表 instance(視覺完整可讀)
- 0% — 戰鬥狀態變異(教學基準應穩定為標準戰,避免新手認知超載)
- 非主軸維度:停留在模板基準

### 失敗後處理(類別:兔子)

- **失敗時**:卡牌消耗,哨兵逃走離開戰場
- **不會變精英回歸**(兔子型無精英化機制)

### 設計合約

- **教玩家**:弱點對應 vs 不對應的次數差距——對應只要 1-2 張,不對應要 4-5 張
- **老手能做到**:純看持械直接對應弱點,1-2 張資源池卡結束戰鬥
- **新手失敗時能理解**:對話會傳遞「方向不對 / 量不夠」的份量感,讓玩家直覺校準「弱點對應有多省」

### 登場時機建議

- **適合出現階段**:早期教學(必出於戰役第一場、首輪連戰開頭)
- **最終連戰適合度**:否(過低需求在最終戰會破壞節奏)
- **不應出現的情境**:中期或後期連戰(此模板是「校準新手手感」用,後期已無教學功能)

**對應敵人設計原則**:第一章(兔子型設計合約)、第三章(原則 #1 持弓/持矛因果)、第七章(兔子失敗代價)、第八章(早期教學定位)。

## 三、巡守(狼型 B,戰鬥狀態變體)

**一句話定位**:中型近戰,定義「戰鬥狀態的影響」——讓玩家體會免費情報(先攻)與資訊封鎖(被襲)的對比

**敵人類別**:**狼(B)** — 威脅普通,失敗會變精英回歸;會記恨並學習

### 層 1 — 模板永久顯性

- 大類識別:中型近戰
- 敵人類別:**狼**
- 弱點範圍:衝擊 / 穿刺 / 燃燒(三種皆可能,視覺完整區分)
- 弱點需求層級範圍:2(中)/ 3(中高)/ 4(高)
- 戰鬥狀態傾向:60% 標準 / 30% 先攻 / 10% 被襲(**此模板的戰鬥狀態是核心變異維度**)

### 層 2 — 次級視覺線索

- 持戰錘/巨槌(雙手鈍器) → 弱點衝擊(對應 A.1)
- 持矛/戟(長柄武器) → 弱點穿刺(對應 A.2)
- 毛皮披風 + 木盾外覆 → 弱點燃燒(對應 A.3「毛皮、皮裘外覆」+「木質結構」)
- 鎧甲層次:皮甲為主 → 中需求(2),鎖鏈+局部金屬 → 中高(3),完整板甲 → 高(4)
- 高處警戒姿勢、瞭望姿態 → 戰鬥狀態傾向先攻(對應 C)
- 隱匿斗篷、低調配色、武器收納 → 戰鬥狀態傾向被襲(對應 C)
- **狼型視覺**:中體型、武器明顯、警戒姿態(對應 D.2)

**覆蓋承諾**:每個實例必持三類弱點視覺其中之一,且戰鬥狀態傾向必有對應視覺暗示。

### 層 3 — 即時狀態

- 戰鬥狀態標籤(本場是標準/先攻/被襲)
- 上一場用情報牌揭露的相關資訊
- 若為先攻變體:**免費揭露本敵的完整需求表**
- 若為精英狀態:UI 標籤「精英化 ⚠」+ 戰鬥紀錄可調出上次資料

### 層 4 — 隱藏狀態(僅情報牌可揭)

完整需求表(從以下 9 個 instance 之一):
- **弱衝擊中**:衝擊 2 / 穿刺 5 / 燃燒 5 / 混合 8
- **弱衝擊中高**:衝擊 3 / 穿刺 6 / 燃燒 6 / 混合 12
- **弱衝擊高**:衝擊 4 / 穿刺 7 / 燃燒 7 / 混合 16
- **弱穿刺中**:衝擊 5 / 穿刺 2 / 燃燒 5 / 混合 8
- **弱穿刺中高**:衝擊 6 / 穿刺 3 / 燃燒 6 / 混合 12
- **弱穿刺高**:衝擊 7 / 穿刺 4 / 燃燒 7 / 混合 16
- **弱燃燒中**:衝擊 5 / 穿刺 5 / 燃燒 2 / 混合 8
- **弱燃燒中高**:衝擊 6 / 穿刺 6 / 燃燒 3 / 混合 12
- **弱燃燒高**:衝擊 7 / 穿刺 7 / 燃燒 4 / 混合 16

### 實例變異規則

此模板的戰鬥狀態本身就是核心變異維度,故偏離預設分布:
- 60% — 主軸:標準戰,需求表 instance 是判斷重點
- 30% — 主軸:先攻變體,免費獲得完整需求表,玩家學習「資訊優勢的價值」
- 10% — 主軸:被襲變體,情報效果減半,玩家學習「資訊封鎖的代價」
- 非主軸維度:需求表 instance 視覺暗示完整呈現

### 失敗後處理(類別:狼)

- **第一次失敗**:該巡守變精英狀態(外觀數值不變,UI 標籤「精英化 ⚠」),排到該連戰末段
- **第二次失敗**:該巡守移出連戰,進入未解決池(玩家可在連戰末選擇追擊或進下連戰時面對)
- **精英狀態失敗**:GAME OVER

### 設計合約

- **教玩家**:戰鬥狀態的影響——免費情報有多寶貴、資訊封鎖有多痛
- **老手能做到**:被襲變體下仍能憑視覺與元知識完成擊殺
- **新手失敗時能理解**:對話會傳遞「先攻變體下你能多確定地擊殺」與「被襲下資訊封鎖有多痛」的對比,讓玩家理解情報的時間性價值

### 登場時機建議

- **適合出現階段**:早期教學(第一輪連戰必出至少 1 隻先攻變體)、中期挑戰(被襲變體開始大量出現)
- **最終連戰適合度**:是,作為連戰節奏的填充
- **不應出現的情境**:絕對首戰(玩家還沒理解戰鬥狀態這個概念)

**對應敵人設計原則**:第一章(狼型設計合約)、第三章(原則 #1 持矛因果、原則 #2 戰鬥狀態為主軸)、第七章(狼型三層失敗代價)。

## 四、重甲衛兵(狼型 B,弱點配對價值)

**一句話定位**:中型需求的物理近戰標準型,定義「弱點對應的投資價值」

**敵人類別**:**狼(B)** — 威脅普通,失敗會變精英回歸;會記恨並學習

### 層 1 — 模板永久顯性

- 大類識別:重甲近戰
- 敵人類別:**狼**
- 弱點範圍:衝擊 / 穿刺(燃燒不在範圍——金屬甲不易燒穿;只能透過混合任意路徑計入)
- 弱點需求層級範圍:2(中)/ 3(中高)/ 4(高)
- 戰鬥狀態傾向:預設標準戰

### 層 2 — 次級視覺線索

- 持斧/持戰錘 → 弱點衝擊(對應 A.1「雙手大型武器」)
- 持矛/持戟 → 弱點穿刺(對應 A.2「長柄武器」)
- 鎧甲縫隙明顯/磨損生鏽 → 弱點需求 2(低,易破)
- 鎧甲整片完整/嶄新 → 弱點需求 3(中)
- 額外肩胸甲強化 → 弱點需求 4(高)
- **狼型視覺**:中體型、武器明顯、警戒姿態(對應 D.2)

**覆蓋承諾**:每個實例必持「斧/錘」或「矛/戟」其中之一,絕不出現「無弱點視覺暗示」的實例。

### 層 3 — 即時狀態

- 戰鬥狀態標籤(本場是標準/先攻/被襲)
- 上一場用情報牌揭露的相關資訊(若該情報牌指向本場目標)
- 若為精英狀態:UI 標籤「精英化 ⚠」+ 戰鬥紀錄可調出上次資料

### 層 4 — 隱藏狀態(僅情報牌可揭)

完整需求表(從以下 6 個 instance 之一):
- **弱衝擊中**:衝擊 2 / 穿刺 5 / 混合 8
- **弱衝擊中高**:衝擊 3 / 穿刺 6 / 混合 12
- **弱衝擊高**:衝擊 4 / 穿刺 7 / 混合 16
- **弱穿刺中**:衝擊 5 / 穿刺 2 / 混合 8
- **弱穿刺中高**:衝擊 6 / 穿刺 3 / 混合 12
- **弱穿刺高**:衝擊 7 / 穿刺 4 / 混合 16

### 實例變異規則

- 80% — 主軸:需求表 instance(視覺線索完整可讀)
- 20% — 主軸:出現「警戒」變體(被襲狀態,情報效果減半)
- 非主軸維度:停留在模板基準(中需求 3,標準戰)

### 失敗後處理(類別:狼)

- **第一次失敗**:該重甲衛兵變精英狀態(外觀數值不變,UI 標籤「精英化 ⚠」),排到該連戰末段
- **第二次失敗**:該重甲衛兵移出連戰,進入未解決池
- **精英狀態失敗**:GAME OVER

### 設計合約

- **教玩家**:弱點對應的投資價值——「對應 = 省一半資源」這個基礎數學
- **老手能做到**:純看持械與鎧甲狀態,完全免情報擊殺(連弱點都不用揭露)
- **新手失敗時能理解**:對話會傳遞「鎧甲太完整、衝擊類型沒對上」的方向感,把錯誤定位在「沒看持械暗示」而不是「我運氣不好」

### 登場時機建議

- **適合出現階段**:早期教學(第一輪連戰必出至少 1 隻)、中期挑戰(精英化變體)
- **最終連戰適合度**:僅限精英化版本,作為對「弱點知識熟練度」的驗收
- **不應出現的情境**:首場戰鬥(玩家還沒學會基本機制)、純情報試探場(此模板強調免情報擊殺)

**對應敵人設計原則**:第一章(狼型設計合約)、第三章(原則 #1 持械因果、原則 #6 視覺覆蓋承諾)、第七章(狼型失敗代價)。

## 五、伏擊者(豹型 C,情報遮蔽)

**一句話定位**:輕甲刺客,定義「資訊不全下的決斷」——預設被襲狀態下,玩家被迫憑視覺與直覺擊殺

**敵人類別**:**豹(C)** — 預設精英,**失敗即 GAME OVER**;「一擊定生死」的氣質

### 層 1 — 模板永久顯性

- 大類識別:輕甲刺客
- 敵人類別:**豹**
- 弱點範圍:穿刺 / 燃燒(衝擊不在範圍——刺客行動敏捷,衝擊難命中;只能透過混合任意路徑計入)
- 弱點需求層級範圍:1(低)/ 2(中)
- 戰鬥狀態傾向:80% 被襲 / 20% 標準(**此模板的核心特性是「資訊封鎖」**)

### 層 2 — 次級視覺線索

- 緊身皮衣裝束 → 弱點穿刺(對應 A.2「皮甲缺乏結構強度,直接被穿透」)
- 多層布料/繃帶纏繞裝束 → 弱點燃燒(對應 A.3「織物外覆」)
- 體型瘦小 / 蹲伏姿態 → 弱點需求 1(低)
- 一般體型 / 直立姿態 → 弱點需求 2(中)
- 隱匿斗篷、暗色低調配色、武器收納 → 戰鬥狀態傾向被襲(預設,對應 C)
- **豹型視覺**:銳利、致命氣質、襲擊預備姿態、暗色配色(對應 D.3)

**覆蓋承諾**:每個實例的服飾必明顯為皮衣或繃帶其中之一。

### 層 3 — 即時狀態

- 戰鬥狀態標籤(本場是標準/先攻/被襲)
- 上一場用情報牌揭露的相關資訊
- 若為被襲變體:標註「情報效果減半」(具體解釋見下)

### 層 4 — 隱藏狀態(僅情報牌可揭)

完整需求表(從以下 4 個 instance 之一):
- **弱穿刺低**:穿刺 1 / 燃燒 3 / 混合 4
- **弱穿刺中**:穿刺 2 / 燃燒 4 / 混合 8
- **弱燃燒低**:穿刺 3 / 燃燒 1 / 混合 4
- **弱燃燒中**:穿刺 4 / 燃燒 2 / 混合 8

被襲狀態下,情報卡只能揭露需求層級(低/中)而非精確需求數字。

### 實例變異規則

此模板的戰鬥狀態以被襲為核心特性,故偏離預設分布:
- 80% — 主軸:被襲狀態下的資訊封鎖,主要靠視覺判讀
- 20% — 主軸:標準變體(罕見,作為對照組)
- 非主軸維度:需求表 instance 視覺暗示完整呈現

### 失敗後處理(類別:豹)

- **失敗即 GAME OVER**——伏擊者是預設精英,本來就「一擊定生死」
- 沒有「曾經是普通」的階段,玩家第一次面對就是高風險
- 視覺暗示要極明顯(刺客的銳利氣質),讓玩家從外觀讀出「不可失敗」

### 設計合約

- **教玩家**:資訊不全下仍可憑視覺判讀完成擊殺——情報投資不是必需,只是輔助
- **老手能做到**:純看裝束類型 + 體型,在被襲狀態下也能 1-2 張資源池卡擊殺
- **新手失敗時能理解**:對話會指出「裝束已透露,被襲不是失敗的藉口」,把錯誤定位在「沒看清裝束」而不是「資訊太少」

### 登場時機建議

- **適合出現階段**:中期挑戰(玩家熟悉基本機制後才適合)、後期驗收
- **最終連戰適合度**:是,作為對「無情報判讀能力」的測試
- **不應出現的情境**:首輪連戰、教學戰(被襲狀態對新手過於懲罰)

**對應敵人設計原則**:第一章(豹型設計合約 + 視覺類別合約)、第三章(原則 #5 失敗代價符合類別合約)、第七章(豹型失敗代價)。

## 六、領主(熊型 D,核心 boss)

**一句話定位**:精英重甲高需求單體,定義「爆發牌與角色牌的投入時機」——需求高到只用資源池卡幾乎不可能

**敵人類別**:**熊(D)** — Boss / 劇情敵人,**失敗即 GAME OVER**;戰役節奏的高潮

### 層 1 — 模板永久顯性

- 大類識別:精英重甲
- 敵人類別:**熊**
- 弱點範圍:衝擊 / 穿刺(燃燒不在範圍——精英重甲完整防護,無有機外覆;只能透過混合任意路徑計入)
- 弱點需求層級範圍:4(高)/ 6(極高)
- 戰鬥狀態傾向:預設標準戰

### 層 2 — 次級視覺線索

- 持巨型雙手戰錘/巨槌 → 弱點衝擊(對應 A.1「雙手大型武器」)
- 持巨型長戟/巨矛 → 弱點穿刺(對應 A.2「長柄武器」)
- 完整重甲 + 標準體型 → 弱點需求 4(高)
- 完整重甲 + 巨大體型 + 額外肩胸甲 → 弱點需求 6(極高)
- **熊型視覺**:壓倒性體型、王者姿態、額外裝飾(對應 D.4)

**覆蓋承諾**:每個實例必持巨型雙手武器其中之一。

### 層 3 — 即時狀態

- 戰鬥狀態標籤
- 上一場用情報牌揭露的相關資訊

### 層 4 — 隱藏狀態(僅情報牌可揭)

完整需求表(從以下 4 個 instance 之一):
- **弱衝擊高**:衝擊 4 / 穿刺 7 / 混合 16
- **弱衝擊極高**:衝擊 6 / 穿刺 9 / 混合 24
- **弱穿刺高**:衝擊 7 / 穿刺 4 / 混合 16
- **弱穿刺極高**:衝擊 9 / 穿刺 6 / 混合 24

### 實例變異規則

- 80% — 主軸:需求表 instance(注意「需求差 2-3 點」對應「需多用 1-2 張爆發牌」)
- 20% — 主軸:極高需求變體(6)是純強度挑戰
- 非主軸維度:預設標準戰

### 失敗後處理(類別:熊)

- **失敗即 GAME OVER**——領主是 Boss / 劇情敵人,代表戰役節奏的高潮點
- 沒有第二次機會,符合「最終一擊」的儀式精神
- 戰役節奏的關鍵位置(每連戰末段 / 最終戰)

### 設計合約

- **教玩家**:爆發牌與角色牌的投入時機——「這隻值不值得用我的關鍵資源?」
- **老手能做到**:精準計算「弱點對應 + 1 張爆發 + 1 張角色」剛好擊殺,不浪費資源
- **新手失敗時能理解**:對話會傳遞「資源量根本不夠、這隻不是工具牌能解決的」的份量感,讓玩家理解「這就是爆發牌存在的原因」

### 登場時機建議

- **適合出現階段**:中期挑戰(極高需求變體)、後期驗收(精英化變體)
- **最終連戰適合度**:是,作為終戰級別的核心對手
- **不應出現的情境**:首輪連戰(玩家還沒累積足夠爆發牌)、純教學戰

**對應敵人設計原則**:第一章(熊型設計合約 + 視覺類別合約)、第三章(原則 #1 持械因果)、第七章(熊型失敗代價)、第八章(後期 boss 定位)。

## 七、連戰配置建議

### 原型階段(2 組連戰)

**連戰 1**(教學基礎):
1. 哨兵(兔子) — 校準弱點對應感
2. 巡守(狼,先攻變體) — 體驗免費情報
3. 重甲衛兵(狼,標準戰) — 弱點對應的投資價值

**連戰 2**(挑戰驗收):
1. 伏擊者(豹,被襲) — 高張力時刻
2. 領主(熊,boss) — 爆發/角色卡的核心使用

### 失敗後的連戰變化

依失敗代價機制(詳見 [遊戲核心系統機制.md](遊戲核心系統機制.md) 第六章):
- 兔子失敗 → 連戰場次 -1(哨兵逃走)
- 狼第 1 次失敗 → 連戰場次 +1(精英排末)
- 狼第 2+ 次失敗 → 進未解決池 → 連戰結束後追擊或替換下連戰
- 豹/熊失敗 → GAME OVER

### 連戰中類別配比建議

| 連戰 | 兔 | 狼 | 豹 | 熊 |
|------|----|----|----|----|
| 早期連戰(教學) | 1-2 | 2-3 | 0 | 0-1(可選) |
| 中期連戰 | 0-1 | 2-3 | 1 | 1 |
| 最終連戰 | 0 | 1-2 | 1-2 | 1(boss) |

兔子比例隨戰役推進降低(從教學淡出);豹熊比例隨戰役推進提升(高張力時刻)。
